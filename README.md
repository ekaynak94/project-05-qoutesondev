## Installation

- Download or clone repo to Wordpress wp-content/themes directory.
- Run the following commands in the directory:

```bash
cd quotes-on-dev
```

```bash
npm install
```

- Enable custom theme 'quotes-on-dev' in Worldpress dashboard.
